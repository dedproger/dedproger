## <a name="introduction"></a> 🔍 ANONNPOの紹介。
これは、デバイスと私生活の完全な匿名化、ハッキング不可能な（理論上は可能な）スマート暗号化プロトコルの基盤構築を専門とする非営利（NoNCommercial）組織です。
法的問題であれサイバーセキュリティであれ、ユーザーを安全にし、不公正の拡大を抑える（結果として生活費が誰にでも手の届くものになる）目的で、あらゆる状況でユーザーを保護します（残念ながら、当組織には犯罪者を処罰する権限はなく、全面的な協力のみを提供します（権限が与えられればハッキングも含む））。
ユーザー自身（生活の監視とパーソナライズ）ではなく、脅威のみを（そして具体的に）コントロールする能力 - これにより、101%の透明性からユーザーは完全な信頼を示すことができます。
すべての製品は、条件のフルスタックを目指しています：モノリシックとモジュラーの同時実現、各ユーザーに対する完全で柔軟な（細部に至るまでの）カスタマイズ、箱から出してすぐに使える匿名性とセキュリティ - つまり、デバイス（ハードウェア）をコントロールするというパラノイアです（2012年以前のThinkPadを使っている人なら私の言っていることがわかるでしょう）。

## <a name="openclosedaccess"></a> 🔄 OpenClousedAccess プロトコル
これはコード標準を保護する実験的なプロトコルです。誰でもコードをチェックし、コードに対して任意の操作を行うことができますが… コードに対するあらゆる操作は、指定されたコード閲覧ポイント以外の場所ではクローズドアクセスになります。完全にオープンなコードは、組織の代表者が指定した特別な場所にあり、行動の監視と管理の下に置かれます。
これが意味するのは == 許可：指定された場所を離れるまでの間、コードに関するあらゆる操作、インターネット上でコードについて意見を述べること（最終決定はされていません（あらゆるコードおよびそのおおよその構造の公開を禁止する / 特に重要でない構造とその例のみ公開を許可する））。訪問回数に制限はありませんが、常識の範囲内です。厳格に禁止：組織の開発物をいかなる形でも持ち出すこと。
おまけの機能は… 時期が来るまで公表しません。
このような対策は、（インターネットはミスを許さないという不注意の例外を除き、偶然ではなく）悪意のある者の容疑者層を絞り込み、ユーザーの安全を確保することができます。
私自身、このような対策はやりすぎだと皆（大部分）に同意しますが、これが現実です。

## <a name="philosophy"></a> 💭 ANONNPOの哲学

活動は、社会（ユーザー）からの完全なフィードバック、市場のニーズに基づいたサービスの改善と販売に向けられており、事前の抽選や「製造しても利益が出ない」という理由でのプロジェクト閉鎖はありません。そして目には目を = ユーザーとANONNPO間のパートナーシッププログラム。その本質は双方の無償の助け合いにあります。ANONNPOは可能な（手の届く）あらゆる問題で支援し、ユーザーは代わりにこの素晴らしいプロジェクト（戦争のない友好的な社会を作り、言葉で和解する）の発展に投資し、発展させます。

## <a name="mission"></a> 🎯 ANONNPOのミッション
第一の課題は、信頼の概念（安全、オープン、無料）に縛られた人間とエコシステムの構築です。あなたのそばにいて、いつでもあなたをサポートしてくれる友人です。厳しすぎたり、許されないことをしたりした親の代わりになることさえあります。子供の頃に莫大なストレスを経験し、社会のレールから外れ、（のけ者の中で）のけ者になってしまった人たち（子供たち）のために。
また、第一の課題は利益経済の完全な崩壊と、公正で正しい価格の実現です。これは、常に利用可能で実質的に無料であり、商品のサービスそのものにのみ料金を請求し、利益を得ないことを意味します。これは必ずしも貧困を意味するものではありません。もし高価な生活を送りたいなら、その方法を見つければいいのです。ANONNPOはあらゆる試みやアイデアをサポートし、「スポンサー」を探すことなく、あらゆるプロジェクトの実現のためにリソースと機能を提供します。会社自身がこれを経験し、長い間何の支援も、すべてがスムーズに進むという確証もなく、すべては希望だけで成り立っていました。
第二の課題 = 例えば、あなたがただの人間ではなく、Cyberpunk 2077のようにインプラントを持っていたり、全身が機械でできているような、CyberPunk時代の変化に社会を準備することです。これは単なる気まぐれではありません。根本的なレベルで多くの病気や人間の問題と戦うための手段です。Cyberpunkの物語の創設者であり主人公として、私はこの種の技術を自分自身で試す最初の人物の1人になります。私は同じくらい巨大な理由から、大きな熱意を持ってこれを見ています。
理解のための短い話：単なるVRゴーグルではなく、別の世界にいるかのような仮想現実ゲーム、Tony Stark（アイアンマン）の軍事的意図のないスーツと技術、物質を通り抜けて他の宇宙へ、そしてすべての世界の境界を越えてその創造の起源（私はそれをメインサーバーと呼んでいます）への飛行、そしてInterstellar (2014) - これらがここにあるのは、これが未来の現実の出来事である可能性があるからです。私は天文学者ではないので追跡はしていませんが、私は常に警戒しており、そのような脅威に対するプランBはあって損はありません。
結論 Human:Cyberpunk Edition == 「第二の課題」= 生存競争の手段であり、最悪の結果に対する早期の準備です。この世界は敵でも味方でもなく、目的を持たずに存在し、様々なメカニズムをシミュレートするために作られています。

## <a name="goals"></a> 📌 ANONNPOの目標
ANONNPOに対するユーザーの信頼 - 完全なオープンソースにより、その安全性を確信し、独立したWhiteHatによってエラーを見つけることができます。その後、WhiteAIHatを使用した脆弱性調査を含む、ANONNPOの全製品の完全なセキュリティのために彼らと協力します。
セキュリティ - 現実世界でのユーザーの完全な匿名化、インターネットやその他のネットワーク上のデータの匿名化 == 私たちは皆、自分自身を示すことを余儀なくされていますが、自分を示すことで常にナイフを突きつけられるリスクを負っています。完全な匿名化により、他人のことを知らずに済むようになり、詐欺師や犯罪のある社会問題について簡単に忘れることができます。これは情報のほんの一部であり、残りはフェーズ2の公開時まで開示する権利を持っていません。
解読や偽造が不可能なセキュリティアルゴリズムとプロトコルの作成により、エコシステムを保護します。ANONNPOは、個人データの収集を含む、ユーザーの明示的な同意なしの監視を完全に非難します（個人データ保護法について（少なくともロシア連邦にはそのような法律が存在します））。個人データ保護法は、ほとんどの場合、追跡されて結果を伴わずにインターネットに投稿できるようなゴミデータではありません。新しいフォーマットである「個人データセキュリティ法」に移行することで、古いフォーマットの法律は時代遅れになり、ネットワーク上の監視の必要性は義務ではなくなります。もちろん、北朝鮮や、正当な理由もなくあらゆるものを禁止する現在の社会のような例外は存在します。
利便性 - エンドユーザーの代わりに常に決定が下される場所です。エンドユーザーは自分が本当に必要で好きなものから選ぶのではなく、会社の創設者/ディレクターの要望に基づいた施しを文字通り提供されています（Apple社 - 利益を犠牲にして品質を作ります）。誰もがカスタムや手頃な価格のことを忘れ、時間が経つにつれて、自分が選ぶものではなく、与えられるものに移行しました。良いものを作る無名でなかなか見つからない愛好家や、あらゆる種類のクールな装飾を作りながら必然的に高価にする（彼らは稼いでいますが）カスタマイザーがたくさんいます。
カスタマイズは、この瞬間から常に偏見よりも価値があるものでなければなりません - あなたが選ぶのであり、あなたのニーズがあなたを選ぶのではありません。
リアリズムは抽象的に見えるべきではない場所です。追加の世界はあってもいいと思います。議論はしません。しかし、世界に不正確な衝突判定、物理演算、世界の動作設定、その他多くのパラメータなどのバグが現れたとき。リアリズムとは現実世界（グラフィックやテクスチャ）についてではなく、3D技術（エンジン、モデリング、TuneVoice（変更）、カスタム音楽、録音メカニズム/世界のカスタマイズ）と組み合わせた現実的なメカニズムと機能についてです。
品質と妥当性の理想的な基準は、アイデアの誕生の最初から始まります。その後のものは問題を解決できない単なる松葉杖です - 私はクリエイター Idealist_Perfectionist としてそう宣言します。質の高い仕事と妥当性が、ミスや問題のない社会を作ります。私たちは皆人間であり... 理論上はミスを犯しますが、実践において理想的な結果をもたらすためにやり直すのです。
